📥 Daten importieren

JSON-Import

Importiere Rechnungen, Kunden, Aufträge und mehr aus einer WerkmeisterPro JSON-Datei. Vorhandene Einträge (gleiche ID) werden nicht überschrieben. Vor dem Import wird automatisch eine Sicherung angelegt.

⚠️
Hinweis
Importierte Datensätze erhalten das Kennzeichen Importiert und sind in Rechnungen, Kunden und Aufträgen farblich markiert. Die Originaldaten werden nicht verändert – nur neue Einträge werden hinzugefügt.

☁ GitHub Sync

Cloud-Backup

Daten werden automatisch in einem GitHub Gist gesichert und geräteübergreifend synchronisiert. Du benötigst einen GitHub-Account und ein persönliches Zugriffstoken (PAT) mit gist-Berechtigung.

Die ID aus der Gist-URL: gist.github.com/username/ID

Erstellen unter: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Scope: gist
